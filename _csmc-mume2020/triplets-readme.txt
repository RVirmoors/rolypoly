The .zip includes 4 .wav files:

- triplets-no : the drum track, as trained normally for 3 iterations
- triplets-no-guitar : one of the guitar takes used in training
- triplets-yes : the drum track, trained w/ triplets for 3 iterations
- triplets-yes-guitar : one of the triplet guitar takes

Also included is the .pt model file for triplets, which can be loaded by roly.py
